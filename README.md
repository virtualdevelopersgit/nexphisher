## Installation :

* `apt update`
* `apt install git -y`
* `git clone git://github.com/htr-tech/nexphisher.git`
* `cd nexphisher`
#### > SETUP : `bash setup`
#### > SETUP [TERMUX] : `bash tmux_setup`
#### > Run : `bash nexphisher`

```
apt update ; apt install git -y ; git clone https://github.com/virtualdevelopersgit/nexphisher.git ; cd nexphisher ; bash setup ; bash nexphisher
```
